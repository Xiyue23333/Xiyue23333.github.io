CHAPTER 01 · SECTION 03

第三节：开始制作自己的 Mod

状态：完整教程 目标：从 0 到可游玩基础内容

页面说明

这一节是“把工程跑起来之后，真正开始做内容”的核心阶段。你会按真实开发顺序，完成物品、创造模式物品栏、方块、掉落表、配方五个步骤，最终得到一个可进入游戏验证的基础教程模组。

本节完成标准

  • 能在游戏中看到并获取 tutorial:crystal、tutorial:crystal_block、tutorial:crystal_ore。
  • 自定义创造模式物品栏可正常显示物品与方块。
  • 矿石掉落与工具等级限制生效（支持精准采集与时运）。
  • 配方支持 9 合 1 与 1 拆 9（crystal ↔ crystal_block）。

要点选择

POINT 01

第一点：制作模组的第一个物品

创建注册类，补齐语言 / 模型 / 贴图资源文件。

 POINT 02

第二点：制作自己模组的创造模式物品栏

注册自定义物品栏，并把你的物品添加进去显示。

 POINT 03

第三点：制作第一个方块

注册方块 + 方块物品，并补齐 blockstates / models / textures。

 POINT 04

第四点：方块掉落物

为矿石编写掉落表，并配置正确工具与石镐等级限制。

 POINT 05

第五点：物品的配方制作

手写工作台配方，完成水晶块的 9 合 1 与 1 拆 9。

建议学习顺序

  1. 先完成 第一点 + 第二点：确保“注册机制 + 显示入口”打通。
  2. 再做 第三点：建立方块与资源文件的目录和命名规范。
  3. 随后完成 第四点 + 第五点：补齐“玩法闭环”（采集与合成）。
  4. 最后跑一次 runClient，逐项验证显示、掉落、配方是否一致。

提醒：Forge 1.20.1 项目建议固定使用 JDK 17。若用更高版本（例如 Java 25）运行 Gradle，可能报 Unsupported class file major version 69。

本节会生成的关键文件

  • src/main/java/com/xiyue/tutorial/common/registry/ModItems.java
  • src/main/java/com/xiyue/tutorial/common/registry/ModBlocks.java
  • src/main/java/com/xiyue/tutorial/common/registry/ModCreativeModeTabs.java
  • src/main/resources/assets/tutorial/*（语言、模型、贴图、blockstates）
  • src/main/resources/data/tutorial/*（loot tables、recipes）
  • src/main/resources/data/minecraft/tags/blocks/*（采掘工具与等级标签）
