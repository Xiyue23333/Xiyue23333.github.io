第 4 章：创建第一个方块

预计时长：45 分钟 难度：入门-进阶

本章目标

完成方块与方块物品注册，并让方块具备可放置、可掉落、可合成的基础行为。

实践步骤

  • 注册 Block 时先用最基础属性，确保稳定可加载。
  • 创建与方块绑定的 BlockItem，确保背包中可拿取与放置。
  • 补齐 blockstate、block model、item model 与 loot table。
  • 添加合成配方，测试生存模式获取流程。

参考代码

public static final RegistryObject STEEL_BLOCK = BLOCKS.register(
  "steel_block",
  () -> new Block(BlockBehaviour.Properties.of().strength(5.0f))
);

public static final RegistryObject STEEL_BLOCK_ITEM = ITEMS.register(
  "steel_block",
  () -> new BlockItem(STEEL_BLOCK.get(), new Item.Properties())
);
