第 3 章：创建第一个物品

预计时长：35 分钟 难度：入门

本章目标

注册一个新物品并在创造模式中可见，补齐语言文件、模型文件与贴图资源。

实践步骤

  • 在注册表中定义 Item，设置创造标签与基础属性。
  • 在 en_us/zh_cn 语言文件中添加翻译键。
  • 补齐 models/item/*.json 与 textures/item/*.png。
  • 进游戏用命令 /give 检查命名空间与注册名是否正确。

参考代码（IDE 视图）

左侧选择文件，右侧可直接编写代码。

public static final DeferredRegister ITEMS =
  DeferredRegister.create(Registries.ITEM, MOD_ID);

public static final RegistryObject COPPER_DUST = ITEMS.register(
  "copper_dust",
  () -> new Item(new Item.Properties())
);
