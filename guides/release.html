第 7 章：打包与发布

预计时长：30 分钟 难度：进阶

本章目标

完成版本号规划、构建产物检查与发布清单准备，让模组可稳定分发给玩家。

实践步骤

  • 使用语义化版本：主版本.次版本.修订号。
  • 构建前执行 clean，确保无旧缓存污染。
  • 检查产物内容：mod id、依赖范围、许可证、更新日志。
  • 同步发布到 GitHub Releases、CurseForge 或 Modrinth。

参考代码（IDE 视图）

左侧选择文件，右侧可直接编写代码。

# 发布前构建
./gradlew clean build

# 产物通常位于
build/libs/your-mod-name-x.y.z.jar
